# HookForge

HookForge is an advanced Android runtime interception framework built on Noxen.

## Features

- Automatic JSON formatting
- Syntax highlighting
- HTTP flow reconstruction
- SSL pinning templates
- Root bypass templates
- Automatic method discovery
- PRE interception mode
- RETURN interception mode
- BOTH interception mode
- Runtime request editing
- Runtime response editing
- OneSpan Device Binding analysis
- Runtime SALT inspection
